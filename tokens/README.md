# Tokens Criados

Esta pasta contém informações sobre tokens criados através da NΞØ SMART FACTORY.

## Estrutura

Cada token deve ter sua própria pasta ou arquivo JSON com:

- Configuração do token
- Endereço do contrato
- Network
- Data de criação
- Informações de deploy

## Formato Recomendado

```json
{
  "name": "Token Name",
  "symbol": "SYMBOL",
  "address": "0x...",
  "network": "polygon",
  "createdAt": "2024-01-01T00:00:00Z",
  "deployer": "0x...",
  "supply": "1000000",
  "price": "0.05"
}
```
