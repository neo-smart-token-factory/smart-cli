# Instruções para Abrir no Word

## ✅ Formato HTML (Recomendado)

Todos os documentos foram criados em formato **HTML** que o Microsoft Word abre **nativamente** e mantém a formatação perfeitamente.

### Como Abrir no Word:

1. **Abra o Microsoft Word**
2. **Arquivo → Abrir**
3. Selecione o arquivo `.html` (não o `.md`)
4. O Word abrirá automaticamente e converterá mantendo toda a formatação
5. **Salve como .docx** quando terminar de assinar

## Alternativa: Canvas.com

1. Abra o Canvas.com
2. Crie novo documento
3. Use "Importar" ou copie/cole o conteúdo HTML
4. O Canvas reconhecerá a formatação HTML

## Vantagens do HTML para Word:

✅ Formatação preservada (títulos, negrito, listas, tabelas)  
✅ Margens e espaçamento corretos  
✅ Fonte Times New Roman (padrão jurídico)  
✅ Tabelas formatadas corretamente  
✅ Sem necessidade de conversores externos  
✅ Funciona em Word 2010, 2013, 2016, 2019, 365
