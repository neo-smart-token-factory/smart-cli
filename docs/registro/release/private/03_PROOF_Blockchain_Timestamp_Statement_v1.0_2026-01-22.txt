O desenvolvimento do projeto teve inicio em 16 de novembro de 2025, conforme comprovado pelo primeiro commit no repositorio GitHub: https://github.com/neomello/neo_smart_factory

A partir de 17 de novembro de 2025, o projeto foi migrado para a organizacao GitHub: https://github.com/neo-smart-token-factory

O projeto consiste em uma infraestrutura de criacao automatizada de protocolos tokenizados em blockchain, utilizando um padrao Factory Modular onde componentes podem ser criados independentemente ou como parte de um protocolo completo.

COMPONENTES PRINCIPAIS:
- FORGE-CORE: Contratos inteligentes (NeoTokenBase, NeoTokenV2, NeoSmartFactory, NeoERC20, NeoERC721, NeoVesting, NeoRewards)
- FORGE-CLI: Interface de linha de comando para criacao e deploy
- FORGE-UI: Interface web e PWA para criacao e gestao
- DOCS: Documentacao tecnica completa (45+ documentos)
- INTERNAL-OPS: Scripts operacionais e simulacao

TECNOLOGIAS:
- Solidity ^0.8.20
- Hardhat
- OpenZeppelin Contracts v5.0
- Redes: Polygon, Base, Arbitrum
- Padroes: ERC20, ERC721, EIP-2612 (Account Abstraction)
